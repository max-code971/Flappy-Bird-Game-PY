## Installation

### Windows

Open Command Prompt and run the following:

```sh
pip install pygame
```

### Mac

Open Terminal and run the following:

```sh
pip3 install pygame
```

### Linux

Debian / Ubuntu

```sh
sudo apt-get install python3-pygame
```
